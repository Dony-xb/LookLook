## 目标
- 在播放页的“点赞、收藏（评论）、分享”图标下显示对应数量，字体大小抽取到资源。
- 在视频右下角显示创建日期（深灰色），字体大小抽取到资源。
- 将 JSON 中的标签追加到视频描述末尾，每个以“#”开头，放在同一段中。

## 数据模型与映射
- 修改 `core/model/Video.kt`：新增可选字段
  - `likesCount: Int?`、`commentsCount: Int?`、`sharesCount: Int?`、`createdAt: String?`
- 修改 `core/repository/VideoRepository.kt` 的 `VideoDto.toDomain()`：
  - 从 `stats?.likesCount`、`stats?.commentsCount`、`stats?.sharesCount` 映射到新字段
  - 从 `createdAt` 映射到新字段；保留原 `sanitizeUrl()` 用于 URL 字段
- 缺失值策略：空值按 0 显示；日期空则不展示

## 资源新增（可视化调整）
- `res/values/integers.xml`
  - `video_action_count_text_size_sp`（图标下数量字体大小，默认 12）
  - `video_date_text_size_sp`（日期字体大小，默认 10）
- `res/values/colors.xml`
  - `video_date_text_color`（深灰，用于日期，默认 `#808080` 或使用主题 onSurface 的 60% 透明）
- 如需统一字重，可复用 `MaterialTheme.typography.bodySmall`

## 播放页 UI 改动（`feature/video/ui/VideoFeedScreen.kt`）
- 右侧垂直操作组 `Column` 中，每个 `IconButton` 下添加一个 `Text` 显示数量：
  - 点赞：`item.likesCount ?: 0`
  - 评论/收藏：按你的 JSON语义，示例用 `item.commentsCount ?: 0`
  - 分享：`item.sharesCount ?: 0`
  - 字体大小：`integerResource(R.integer.video_action_count_text_size_sp).sp`
  - 颜色：白色或主题 onSurface，与你当前风格一致
- 右下角日期：
  - 在页面根 `Box` 内、靠近底部进度条上方或同层级添加 `Text`
  - 内容：`item.createdAt`（如需格式化可后续扩展为 `yyyy-MM-dd`）
  - 样式：`fontSize = integerResource(R.integer.video_date_text_size_sp).sp`、`color = colorResource(R.color.video_date_text_color)`、`Modifier.align(Alignment.BottomEnd).padding(end = 16.dp, bottom = 8.dp)`
- 标签与描述：
  - 构造：`descWithTags = (item.description ?: "") + " " + item.tags.joinToString(" ") { "#" + it }`
  - 显示：底部 `Column` 第二行使用 `descWithTags`
  - 若无标签或描述为空，保证字符串安全拼接

## 目标概述

* 抽取主页封面标签字体大小与标签内边距（文字与边框的间距）为资源，支持可视化调整。

* 播放页底部文案分层：上方显示用户名（以@开头），下方显示视频描述；两者字体大小可资源化调整。

* 播放页点击暂停时显示暂停标识；长按切换二倍速并显示 2x 标识；再次长按取消二倍速。

* 播放页底部增加极细进度条，紧贴屏幕底部，参考抖音样式。

## 文件与位置

* 主页：`app/src/main/java/com/looklook/feature/home/ui/HomeScreen.kt`

* 播放页：`app/src/main/java/com/looklook/feature/video/ui/VideoFeedScreen.kt`

* 资源（可视化修改入口）：`app/src/main/res/values/dimens.xml`、`app/src/main/res/values/integers.xml`、`app/src/main/res/values/colors.xml`

## 资源抽取与新增

* 主页封面标签

  * 新增 `integers.xml`: `home_tag_text_size_sp`

  * 已有标签内边距：`home_tag_padding_h`、`home_tag_padding_v`（作为文字与标签边框的间距），保留并继续使用

* 播放页文字与指示

  * 新增 `integers.xml`: `video_username_text_size_sp`、`video_description_text_size_sp`、`video_speed_indicator_text_size_sp`

  * 新增 `dimens.xml`: `video_progress_height`（进度条高度，如 2dp）

  * 新增 `colors.xml`: `video_progress_color`、`video_progress_track_color`（进度条前景/轨道颜色）、`video_pause_indicator_tint`、`video_speed_indicator_tint`

## 主页实现变更

* `HomeScreen.kt`

  * 标签文字 `Text`：使用 `fontSize = integerResource(R.integer.home_tag_text_size_sp).sp`

  * 标签内边距仍使用 `home_tag_padding_h/v`（你可以在 Resource Manager 可视化调整）

## 播放页实现变更

* 文案布局

  * 底部文案改为两行：第一行用户名（前缀 @），第二行视频描述

  * 字体大小使用 `integerResource(...).sp`，资源为 `video_username_text_size_sp` 与 `video_description_text_size_sp`

* 暂停标识

  * 点击屏幕时：切换播放/暂停，并在暂停态于中心显示半透明暂停图标（Material Icons），恢复播放即隐藏

* 二倍速标识与交互

  * 长按屏幕：若当前为 1x → 切到 2x 并显示 “2x” 标识；若当前为 2x → 切回 1x 并隐藏标识

  * 采用 `exoPlayer.playbackParameters = PlaybackParameters(speed)` 并同步本地状态用于 UI 显示

* 底部进度条

  * 紧贴屏幕底部的极细 `LinearProgressIndicator` 或自绘进度条，高度由 `video_progress_height` 控制

  * 进度由 `currentPosition / duration` 计算，每 100ms 左右刷新（在 `LaunchedEffect(exoPlayer)` 中以非 UI 线程收集）

## 事件与状态

* 在 `VideoFeedScreen.kt` 添加本地 UI 状态：`isPaused`、`isDoubleSpeed`、`progress`（0f-1f）

* 在 `Player.Listener` 中同步 `ready/playing` 等状态；在 `pointerInput` 的 `detectTapGestures` 使用 `onTap` 切换暂停/播放、`onLongPress` 切换倍速

## 验证与回归

* 运行 `assembleDebug` 验证编译通过

  * 长按切换到 2x 显示“2x”标识，再次长按取消二倍速
